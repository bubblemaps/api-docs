---
title: "Get Batch Tokens Metadata"
openapi: "POST /v0/tokens/metadata"
---

<Info>**Credit cost:** 0.1 per token.</Info>

Get metadata for a batch of tokens. Token list is passed as query body.

The parameter `return_token_stats` will get you some stats about the tokens. Costs 0.5 credit per token instead of 0.1.
